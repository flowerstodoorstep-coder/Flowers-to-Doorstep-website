# Flowers to Doorstep Storefront & Order Hub

A localized, hyper-fast mobile web application tailored specifically for managing divine fresh bloom distribution and instant catalog orchestration in Gowlidoddi. 

## Features
- **Live Inventory Synchronization:** Direct data integration powered by an optimized Firebase Realtime Node data structure.
- **Defensive Mobile Event Scope Routing:** Prevents interaction delay and scope drops across older Safari and Android WebView instances.
- **Automated WhatsApp Routing Engine:** Instant order breakdown formatting automatically pre-fills standard customer messaging parameters.
- **Secure Access Validation Portal:** Integrated, password-guarded catalog controls to manipulate prices and check active system records instantly.
